Supply Chain Monitoring with Python and AI — Detect Changes Before They Hit You

By Mahika Jadhav · May 31, 2026 · 5 min read

Supply chain disruptions don't announce themselves. Lead times quietly extend. Stock availability drops. Supplier notices appear on obscure portal pages. By the time you find out through normal channels, you're already behind.

An AI monitoring agent watching your suppliers' websites can surface these signals early — but only if it's built to distinguish real changes from noise, and only fire alerts when something actually shifts.

What to monitor

  • Supplier product pages (availability, lead times, minimum order quantities)
  • Shipping and logistics provider service notices
  • Component manufacturer announcements (EOL notices, allocation alerts)
  • Port and logistics status pages
  • Government trade and tariff pages for relevant commodities

import indra
from anthropic import Anthropic

agent = indra.init(brightdata_api_key="your-bd-key")
client = Anthropic()

def analyze_supply(goal, inputs, *_):
    content = inputs.get("diff") or inputs.get("content", "")
    return client.messages.create(
        model="claude-haiku-4-5", max_tokens=512,
        messages=[{"role": "user", "content": f"{goal}\n\n{content}"}]
    ).content[0].text

SUPPLIERS = [
    {
        "name": "Main Component Supplier",
        "url": "https://supplier-a.com/components/widget-a",
        "question": "Did stock availability, lead time, or pricing change? What's the current status?"
    },
    {
        "name": "Backup Supplier",
        "url": "https://supplier-b.com/availability",
        "question": "Any changes to availability or lead times for electronic components?"
    },
    {
        "name": "Logistics Provider",
        "url": "https://logistics-co.com/service-alerts",
        "question": "Are there any new service disruptions, delays, or surcharges?"
    },
]

for supplier in SUPPLIERS:
    result = agent.watch(
        url=supplier["url"],
        question=supplier["question"],
        generation_fn=analyze_supply,
    )

    if result.changed:
        print(f"⚠️  SUPPLY CHANGE: {supplier['name']}")
        print(result.insight)
        alert_procurement_team(supplier["name"], result.insight)

Monitoring commodity and tariff pages

# Track regulatory changes that affect sourcing
result = agent.watch(
    url="https://hts.usitc.gov/",  # US tariff schedule
    question="Did tariff rates change for semiconductor components (chapters 85-86)?",
    generation_fn=analyze_supply,
)

if result.changed:
    alert_finance_team("Tariff change detected", result.insight)

Early warning system

import datetime

# Run every 4 hours via cron: 0 */4 * * *
results = []

for supplier in SUPPLIERS:
    result = agent.watch(url=supplier["url"], question=supplier["question"], generation_fn=analyze_supply)
    results.append({"supplier": supplier["name"], "changed": result.changed, "insight": result.insight})

changes = [r for r in results if r["changed"]]

if changes:
    urgency = "HIGH" if any("out of stock" in r["insight"].lower() or "lead time" in r["insight"].lower() for r in changes) else "MEDIUM"
    send_supply_alert(urgency, changes)
