SERP Monitoring in Python — Track Rankings with LLM Intelligence

By Mahika Jadhav · May 31, 2026 · 5 min read

Knowing when your competitors move in search rankings — or when your brand appears in new contexts — gives you time to respond. But scraping Google SERPs has two problems: Google blocks it aggressively, and SERPs change slowly — paying an LLM to read stable results daily is wasteful.

Indra's search_watch() solves both: Bright Data handles Google's bot detection, and change detection fires the LLM only when rankings actually shift.

Track your own rankings

import indra
from anthropic import Anthropic

agent = indra.init(brightdata_api_key="your-bd-key")
client = Anthropic()

def analyze_serp(goal, inputs, *_):
    content = inputs.get("diff") or inputs.get("content", "")
    return client.messages.create(
        model="claude-haiku-4-5", max_tokens=512,
        messages=[{"role": "user", "content": f"{goal}\n\n{content}"}]
    ).content[0].text

# Track your brand
result = agent.search_watch(
    query="your product name",
    question="What's our ranking position? Did any new reviews or competitor mentions appear?",
    generation_fn=analyze_serp,
)

if result.changed:
    print(f"SERP shifted: {result.insight}")

Competitive SERP intelligence

target_keywords = [
    "best llm caching library",
    "langchain execution cache",
    "reduce openai api costs python",
    "competitor name alternatives",
]

for keyword in target_keywords:
    result = agent.search_watch(
        query=keyword,
        question="Who's ranking in the top 5? Did any positions change since last check? Did we gain or lose positions?",
        generation_fn=analyze_serp,
    )

    if result.changed:
        print(f"\nRanking shift: '{keyword}'")
        print(result.insight)

Running a daily SERP digest

keywords = load_tracked_keywords()
changes = []

for kw in keywords:
    result = agent.search_watch(query=kw, question="Any ranking changes or new competitors?", generation_fn=analyze_serp)
    if result.changed:
        changes.append({"keyword": kw, "change": result.insight})

stats = agent.get_stats()
print(f"Keywords tracked: {len(keywords)}")
print(f"SERP changes detected: {len(changes)}")
print(f"LLM calls: {stats['llm_calls_fired']} (vs {len(keywords)} naive)")

if changes:
    send_seo_digest(changes)
SERP monitoring requires Bright Data to bypass Google's bot detection. Without it, requests from a static IP are blocked within hours. Indra abstracts this — the same search_watch() call works regardless of the target search engine.
